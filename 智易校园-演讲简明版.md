# 🎓 智易校园 —— 7/13讲解

> 基于 AI 辅助审核与闭环生态的校园二手交易平台

---

## 一、项目定位

面向高校学生的二手/需求交易平台，一句话概括：**AI 审核 + 担保交易 + 内置聊天 + 成长体系**。

| 维度 | 亮点 |
|------|------|
| 🤖 AI 智能审核 | 接入大模型，自动识别违禁内容 + 智能提取商品标签 |
| 🔒 担保交易 | 平台托管资金，收货后打款，保障双方权益 |
| 🌱 成长体系 | 经验值 + 等级 + 信誉分，激励诚信交易 |
| 💬 内置聊天 | 站内沟通，无需跳转第三方 IM |
| 🛡️ 风控惩罚 | 警告 → 限时封禁 → 永久封号 |

---

## 二、模块划分（4 人小组）

| 成员 | 模块 | 核心内容 |
|------|------|----------|
| A | 用户认证与成长体系 | 注册登录、密保找回、等级经验、违规封禁 |
| B | AI 赋能的内容发布 | 商品发布、AI 审核、动态打标、分类管理 |
| C | 交易大厅与社交互动 | 商品大厅、搜索筛选、内置聊天、排行榜 |
| D | 虚拟钱包与超管控制台 | 钱包充值、担保交易、数据大盘、客服系统 |

---

## 三、各模块功能一览

### 模块一：用户认证与成长体系（A）

| 功能 | 一句话说明 |
|------|-----------|
| 🔐 注册登录 | 学号 + 密码注册，BCrypt 加密，JWT 认证 |
| 🔑 密码找回 | 预设密保问题验证，答案 BCrypt 存储 |
| 👑 角色划分 | USER / ADMIN，注解 `@RoleRequired` 鉴权 |
| 📈 成长体系 | 交易得经验，累积升级（Lv.1~5），等级徽章展示 |
| ⚠️ 违规封禁 | 警告 / 限时封禁 / 永久封号，到期自动恢复 |

**额外扩展**：修改密码、注销账号（软注销，有进行中订单禁止）

---

### 模块二：AI 赋能的内容发布（B）

| 功能 | 一句话说明 |
|------|-----------|
| 📂 商品分类 | 8 个预设大类，AI 动态生成小标签 |
| 📝 双向发布 | 可发"出售"和"求购"两种帖子 |
| 🖼️ 图片上传 | 1~9 张，jpg/png/webp，单张 ≤5MB |
| 🤖 AI 智能审核 🔥 | 调大模型审核内容合规性 + 自动提取标签 |
| 📦 商品管理 | 编辑、下架、重新上架、软删除 |

**AI 审核流程**：用户提交 → 组装 Prompt → 调 AI → 违规拦截 / 合规打标发布。超时则默认放行 + 标记人工复核。

---

### 模块三：交易大厅与社交互动（C）

| 功能 | 一句话说明 |
|------|-----------|
| 🏠 商品大厅 | 卡片流展示，默认随机排序，支持最新/价格排序 |
| 🔍 搜索筛选 | 关键词搜标题+标签+描述，大类+价格区间筛选 |
| 📋 商品详情 | 图片轮播、卖家信息+等级、AI 标签、收藏/联系/购买 |
| ❤️ 收藏 | 收藏/取消切换，个人中心查看收藏列表 |
| 🏆 爆款榜单 | Top 10 收藏最高商品，LEFT JOIN + GROUP BY |
| 💬 内置聊天 🔥 | 前端定时轮询 2-3s，买家→卖家、用户→客服 |

---

### 模块四：虚拟钱包与超管控制台（D）

**普通用户端：**

| 功能 | 一句话说明 |
|------|-----------|
| 💰 虚拟钱包 | 余额查看、模拟充值（¥0.01~10000）、资金流水 |
| 🔒 担保交易 🔥 | 买家付款→资金冻结→确认收货→卖家收款，全程事务 |
| 📦 订单管理 | "我买的"（确认收货/取消）、"我卖的" |

**管理员端：**

| 功能 | 一句话说明 |
|------|-----------|
| 📊 数据大盘 | 注册用户/活跃商品/今日交易额/拦截违规 四张统计卡片 |
| 🛡️ 违规审核 | 查看 AI 拦截记录→确认封禁/误判放行 |
| 💬 客服收件箱 | 统一处理所有用户客服会话 |
| 🔧 强制管理 | 强制下架商品、强制重置密码 |

## 四、核心技术

### 🔥 三大重点

1. **AI 智能审核** — 大模型 API 自动识别违禁品/代考代写/虚假信息，自动提取商品标签，超时降级放行不阻塞用户
2. **担保交易** — `@Transactional` 保证资金扣款→冻结→解冻全流程数据一致性，是展示 ACID 能力的绝佳场景
3. **内置聊天** — 轮询拉取消息，买家卖家站内沟通，无需 WebSocket 也能演示

### 🏗️ 技术栈

| 层 | 选型 |
|------|------|
| 后端 | Spring Boot 3.2 + MyBatis-Plus + MySQL 8.0 |
| 认证 | JWT + Token 版本机制 |
| 前端 | Vue 3 + Element Plus + Vite |
| AI | 通义千问 / 文心一言 API |
| 高并发 | Caffeine 本地缓存 + HikariCP 连接池 |
