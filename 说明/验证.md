
  问题二：怎么启动

  按以下步骤操作：

  1. 启动后端

  打开终端，进入后端目录，设置环境变量后启动：

  cd zhiyi-campus/backend

  # PowerShell 设置环境变量（正确写法）
  $env:ANTHROPIC_AUTH_TOKEN="sk-6778488a249e41c6a0bd8871074f97b7"
  $env:MYSQL_PASSWORD = "wangke3398"
  $env:JWT_SECRET="zhiyi-campus-2026-this-is-a-very-long-secret-key!!";
  # 然后启动后端
  mvn spring-boot:run

  启动成功后会看到 Tomcat started on port 8080。

  2. 启动前端

  另开一个终端：

  cd zhiyi-campus/frontend

  # 第一次需要安装依赖（之后不需要）
  npm install

  # 启动开发服务器
  npm run dev

  启动后会显示 http://localhost:3000。前端 Vite 已配置好代理，/api 请求会自动转发到 localhost:8080。

  3. 登录测试

  - 管理员：admin / admin（密码是 BCrypt 哈希，需要先在本地重置。或者直接注册一个普通用户测试钱包功能）

  4. 验证 4.1
